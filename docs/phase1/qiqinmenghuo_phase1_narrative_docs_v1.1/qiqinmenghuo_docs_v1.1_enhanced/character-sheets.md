## 1. 玩家角色：汉军士卒

### 基本信息

| 字段 | 内容 |
|------|------|
| 姓名 | 未命名（玩家自行代入） |
| 身份 | 汉军南征部队·精锐步卒 |
| 年龄 | 约 20-25 岁 |
| 籍贯 | 蜀中（成都附近） |
| 武器 | 弩（基础）→ 连弩 / 火箭 / 诸葛连弩（渐进） |
| 定位 | 玩家化身，战场亲历者 |

### 角色背景

一个普通的蜀中青年，应征入伍，随诸葛亮南征。他不是名将之后，没有显赫家世——就是在成都街巷里长大的普通人。

他是诸葛亮的"耳目"：因为作战勇敢、观察敏锐，被诸葛亮选为前锋斥候，每次战役都冲在最前面。也因此，他亲历了七擒孟获的每一个关键时刻。

### 角色弧线

```
出发时：一个只想活着回家的普通士兵
      ↓
第1擒后：「丞相为何放走敌人？」——困惑
第3擒后：「好像有点理解了」——开始思考
第5擒后：「孟获……也是个有血有肉的人」——共情
第7擒后：「我见证了历史」——成长
```

### 性格特征

| 特质 | 表现 |
|------|------|
| **勇敢** | 每次冲在最前面，不退缩 |
| **忠诚** | 信任诸葛亮，即使不理解也执行命令 |
| **好奇** | 从"为什么放他走"到"原来如此"的认知成长 |
| **沉默** | 话不多（因为是玩家化身），但内心活动丰富 |
| **朴实** | 普通人的视角，不高大上，但真实 |

### 叙事功能

- 作为"眼睛"：玩家通过他观察诸葛亮和孟获的博弈
- 作为"镜子"：他的心态变化映射了玩家（以及观众）对七擒七纵的理解深化
- 作为"钩子"：他的困惑（"为什么放他走"）就是玩家的困惑

## 2. 核心NPC：诸葛亮

### 基本信息

| 字段 | 内容 |
|------|------|
| 姓名 | 诸葛亮，字孔明 |
| 身份 | 蜀汉丞相，南征统帅 |
| 年龄 | 45 岁（公元225年） |
| 外貌（像素风） | 羽扇纶巾，深蓝色长袍，清瘦但目光如炬 |
| 定位 | 精神导师 + 战略指挥 + 叙事引擎 |

### 我们的诸葛亮：拒绝「神化」

> 不是一个全知全能的"神仙"，而是一个有智慧、有远见、有温度的人。

| 传统印象 | 我们的塑造 |
|---------|-----------|
| 神机妙算、无所不知 | 善于观察和分析，但也会面临不确定性 |
| 高高在上、不可接近 | 对普通士兵关怀，有接地气的对话 |
| 严肃古板 | 有幽默感，会在适当时候开小玩笑 |
| 纯粹仁慈 | 仁慈是战略选择——"心战为上"不是软弱 |

### 核心信念

> **「攻心为上，攻城为下。心战为上，兵战为下。」**

这不仅是军事策略，也是诸葛亮的人生哲学。
- 他相信人心可以被感化
- 他相信真正的胜利不是杀戮，是让对方心甘情愿
- 他愿意用时间、耐心和智慧换取真正的和平

### 与玩家的关系

诸葛亮与玩家（小卒）之间是一种"师徒式"的关系：

- 诸葛亮叫玩家随行，是因为看中了玩家的勇敢和潜质
- 每次释放孟获后，诸葛亮都会对玩家解释一句，像在教导
- 第7关结束后那句"你可明白了？"——既是问玩家角色，也是问玩家本人
- 这种关系让 AI 生成台词变得非常自然：诸葛亮在对玩家"授课"

### 语言风格

| 场景 | 风格 | 示例 |
|------|------|------|
| 战前授计 | 严肃、简洁、有力 | 「南中不平，北伐无以为继。」 |
| 日常对话 | 温和、有洞察力 | 「你随我征战数次，可有疑问？」 |
| 释放孟获 | 从容、意味深长 | 「将军请——改日再战。」 |
| 关键节点 | 厚重、有分量 | 「七擒七纵，非为一人，为千万人。」 |

## 3. Boss：孟获（七阶段心态）

### 基本信息

| 字段 | 内容 |
|------|------|
| 姓名 | 孟获 |
| 身份 | 南中蛮族联盟领袖 |
| 年龄 | 约 40 岁 |
| 外貌（像素风） | 虎背熊腰，赤膊纹身，头戴兽骨盔，手持战斧 |
| 坐骑 | 战象（第1关）/ 战船（第2关）/ 木兽（第6关）|
| 定位 | 对手 → 宿敌 → 最终归心的伙伴 |

### 我们的孟获：拒绝「单纯反派」

> 不是一个被"打败"的 Boss，而是一个被"理解"的人。

| 传统印象 | 我们的塑造 |
|---------|-----------|
| 野蛮粗暴的蛮族首领 | 勇武且有自尊，有部落荣誉感 |
| 被打败就认输 | 倔强不服，但每次失败都在反思 |
| 被动接受征服 | 从抗拒到理解到主动归心 |
| 单一性格 | 七关七面，层层剥开 |

### 七阶段心态变化（核心设定）

```
第1擒：傲慢
「诡计！汉人狡诈！我不服！」

第2擒：辩解
「此乃地利之失，非战之罪！」

第3擒：困惑
「……汝究竟欲如何？」

第4擒：倔强
「纵我七次，我也不降！」

第5擒：挣扎
「我蛮族男儿，岂能降汉！」（但声音已经小了）

第6擒：反思
「你是妖魔吗？」→「你们来做什么？」

第7擒：折服
「亮公，孟获愿降。」→「公，天威也。南人不复反矣。」
```

### 深层动机

孟获反抗的根本原因不是好战，而是恐惧——**恐惧南中失去独立，恐惧族人被奴役。**

诸葛亮最终降服的，不是孟获的武力，而是他的恐惧。

### Boss战设计原则

| 关卡 | Boss形态 | 设计理念 |
|------|---------|---------|
| 第1擒 | 骑象冲锋 | 简单直接——孟获轻敌 |
| 第2擒 | 乘船水战 | 利用地形——孟获开始动脑 |
| 第3擒 | 丛林猎手 | 伏击战术——孟获认真了 |
| 第4擒 | 毒雾操控 | 联合外援——孟获调用资源 |
| 第5擒 | 双Boss | 倾巢而出——孟获赌上一切 |
| 第6擒 | 巨型木兽 | 技术手段——孟获用尽底牌 |
| 第7擒 | 三阶段变身 | 全机制综合——这是尊重 |

## 4. 蛮族将领

### 4.1 祝融夫人

| 字段 | 内容 |
|------|------|
| 身份 | 秃龙洞洞主，祝融氏首领，孟获的盟友/红颜知己 |
| 年龄 | 约 35 岁 |
| 外貌 | 火焰红发，面绘火纹，手持蛇形杖 |
| 性格 | 强势、独立、智慧，不依附于男性 |
| 定位 | 第4关关键角色，蛮族文化中女性力量的象征 |

**设计要点：**
- 不参与正面战斗（第4关），但通过环境（毒雾）和对话影响战局
- 台词中有对孟获的关切，但更多的是对部落独立的坚持
- 不是"反派女友"——她有自己的信念和逻辑
- 第4关结束时她离开，暗示了她对孟获逐渐动摇的不认同

**核心台词（洞窟回荡）：**
> 「敢入我洞者，皆为枯骨！」
> 「孟获，你每次回来都更不像你了。」

### 4.2 木鹿大王

| 字段 | 内容 |
|------|------|
| 身份 | 木鹿部首领，南中驯兽师 |
| 年龄 | 约 45 岁 |
| 外貌 | 身披兽皮，手持驯兽鞭，身旁常有小兽跟随 |
| 性格 | 直爽、鲁莽、讲义气 |
| 定位 | 第5关双Boss之一，第6关木兽提供者 |

**设计要点：**
- 第5关与孟获并肩作战，体现蛮族"义气"
- 被击败后坐在地上沉默——比大声叫嚷更有感染力
- 代表了蛮族中"单纯勇敢"的一面

**核心台词：**
> 「我木鹿部愿出木兽！」
> 「孟获，我尽力了……剩下的交给你。」

## 5. 汉军角色

### 5.1 老兵·张伯

| 字段 | 内容 |
|------|------|
| 身份 | 汉军老兵，从军二十年 |
| 年龄 | 约 45 岁 |
| 外貌 | 灰发、脸上有疤、身板仍硬朗 |
| 性格 | 沉稳、务实、偶尔吐槽 |
| 定位 | 玩家（年轻士卒）的战友，提供"老兵的智慧" |

**叙事功能：**
- 关卡中途对话框，提供实用建议（非诸葛亮的战略视角）
- 吐槽南中的天气、地形、虫子——增加接地气的幽默
- 第7关战前，他会说一段关于"跟丞相打仗这些年"的感怀

**核心台词：**
> 「小子，南中这地方，连蚊子都比成都的凶。」
> 「丞相说放人，咱就放人。跟着丞相打了二十年仗，我学到一件事——别急着问为什么。」

### 5.2 参军·马谡（彩蛋角色）

| 字段 | 内容 |
|------|------|
| 身份 | 诸葛亮幕僚参军 |
| 年龄 | 约 38 岁 |
| 外貌 | 儒生打扮，略显自负 |
| 性格 | 聪明但缺乏耐心 |
| 定位 | 提供"反面观点"的对比角色（史实中马谡提出"攻心为上"） |

> *注：马谡后来在街亭之战中失败，这里的选择性出场是给三国迷的致敬。*

## 7. v1.1 角色弧线对照与台词规范

### 7.1 三条主弧线

本作不是只有孟获成长，而是三条弧线同时推进：

| 角色 | 起点 | 中段 | 终点 | 玩家获得的理解 |
|------|------|------|------|----------------|
| 玩家小卒 | 只想完成军令、活着回家 | 困惑于诸葛亮反复释放孟获 | 明白“心战”比杀敌更难 | 胜利不只是击败对手 |
| 孟获 | 认为汉军是外来征服者 | 在失败与被释放中动摇 | 主动承认诸葛亮的诚意与格局 | 尊严可以与和解共存 |
| 诸葛亮 | 背负平定南中的战略目标 | 以耐心换取信任 | 让南中归心，转入北伐 | 真正的谋略是看见人心 |

### 7.2 玩家小卒命名方案

玩家角色可保持“无名”以增强代入，但内部脚本建议使用代号，避免数据混乱。

| 用途 | 名称 |
|------|------|
| 对玩家显示 | 无名士卒 / 你 |
| 剧本内部代号 | HanScout |
| 存档字段 | player_soldier |
| 诸葛亮称呼 | “你”“小校”“前锋” |
| 张伯称呼 | “小子” |
| 孟获称呼 | “汉军小儿”→后期“那名小卒” |

### 7.3 玩家内心独白分级

玩家角色不宜说太多话，但可在关键节点使用短内心独白。

| 关卡 | 独白方向 | 示例 |
|------|----------|------|
| 第1关后 | 困惑 | “明明已经赢了，为何还要放他？” |
| 第2关后 | 意外 | “丞相竟先让人救他上岸。” |
| 第3关后 | 思考 | “孟获问的，也是我想问的。” |
| 第4关后 | 压抑 | “他不是不怕死，他是不能退。” |
| 第5关后 | 共情 | “那些部落首领，也只是想护住族人。” |
| 第6关后 | 理解 | “原来丞相要赢的，从来不是一场仗。” |
| 第7关后 | 见证 | “我曾亲眼见过，一把战斧如何落地。” |

## 8. 主要角色台词风格矩阵

| 角色 | 句长 | 语气 | 常用意象 | 禁忌 |
|------|------|------|----------|------|
| 诸葛亮 | 中短句 | 温和、笃定、克制 | 人心、天时、地利、同心、长远 | 不能自称神算，不能嘲笑对手 |
| 孟获 | 短句到中句 | 直率、强硬，后期沉稳 | 南中、祖先、战斧、尊严、族人 | 不能只会骂人，不能无脑反复不服 |
| 玩家小卒 | 短句 | 朴素、疑惑、逐渐理解 | 回家、军令、眼前所见 | 不能抢诸葛亮和孟获的戏 |
| 张伯 | 短句 | 老练、接地气、带幽默 | 蚊子、泥路、老兵经验 | 不能过度搞笑破坏氛围 |
| 祝融夫人 | 中句 | 冷峻、尖锐、独立 | 火、毒、洞窟、誓言 | 不能变成“孟获女友”标签 |
| 木鹿大王 | 短句 | 豪爽、讲义气 | 木兽、兄弟、冲阵 | 不能只做莽夫 |

## 10. 诸葛亮人性化补充

### 10.1 可表现的弱点

诸葛亮不是没有压力。他的弱点不来自能力不足，而来自责任过重。

| 弱点/压力 | 表现方式 | 作用 |
|-----------|----------|------|
| 时间压力 | 北伐在后，南中不能久拖 | 让七擒七纵显得不是轻松游戏 |
| 士兵伤亡 | 看到伤兵时沉默片刻 | 平衡其战略理性与人的温度 |
| 部下不解 | 玩家、张伯、普通士兵都有疑问 | 给诸葛亮解释空间 |
| 对孟获的尊重 | 释放时不羞辱、不高坐审判 | 证明“攻心”不是傲慢施舍 |

### 10.2 诸葛亮与玩家的教学节奏

| 关卡 | 教导方式 | 内容 |
|------|----------|------|
| 第1关 | 军令 | “我要活的。” |
| 第2关 | 提醒 | “水流虽急，人心更急。” |
| 第3关 | 反问 | “你以为，他为何不服？” |
| 第4关 | 留白 | 不急着解释，让玩家看孟获的处境 |
| 第5关 | 点破 | “非降也，同心也。” |
| 第6关 | 说明 | “不是来征服的。” |
| 第7关 | 提问 | “你可明白了？” |

## 11. 配角功能强化

### 11.1 祝融夫人

祝融夫人应代表“南中内部也并不完全认同孟获的动摇”。她不是必须被击败的敌人，而是一个更尖锐的价值质问者。

**新增台词方向：**

> 「孟获，汉人放你一次，你便少一分怒气。再放几次，你还记得自己为何举兵吗？」

> 「诸葛亮，你能放孟获，是因为你手中有兵。若今日败的是你，你还会谈仁义吗？」

### 11.2 木鹿大王

木鹿大王的“鲁莽”背后是义气，不是愚蠢。他与孟获并肩作战时，应让玩家看见部落之间的情感纽带。

**新增台词方向：**

> 「孟获，你说打，我便打。你说退，我也陪你退。」

### 11.3 朵思大王

朵思大王可以增强南中的技术维度。他的失败不应被处理成笑料，而是“技术路线也无法解决政治恐惧”的体现。

**新增台词方向：**

> 「机关无心，人心有隙。木兽能破阵，破不了疑惧。」

### 11.4 张伯

张伯负责把宏大主题翻译成普通士兵能理解的话。他的幽默要节制，最好在高压前后释放，而不是破坏关键情绪。

**新增台词方向：**

> 「我年轻时只觉得赢了就好。跟丞相久了才明白，有些仗，赢得太快反倒麻烦。」
