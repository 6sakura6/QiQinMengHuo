## v1.1 修订目标

本版在 v1.0 的基础上重点补强三件事：

1. **定义更清晰**：把“魂斗罗式爽快射击”和“七擒七纵叙事循环”拆成可落地的设计原则，避免概念停留在口号。
2. **叙事更闭环**：让诸葛亮、孟获、玩家小卒三条成长线相互映照，确保每一关的战斗机制都能回答“为什么放走孟获”。
3. **开发更可执行**：补充数据结构、验收标准、风险规避、Phase 2 原型范围，方便后续直接进入 Phaser 原型制作。

## 叙事主线

```
诸葛亮率汉军南征 → 七次擒获孟获 → 七次释放 → 孟获从傲慢到折服 → 南中归心

核心叙事钩子：为什么打赢了还要放人？ → 答案随7关逐步揭晓
```

## 孟获心态变化弧线

```
第1擒：傲慢 ──→ 第2擒：辩解 ──→ 第3擒：困惑 ──→ 第4擒：倔强
                                                ↓
第7擒：折服 ←── 第6擒：反思 ←── 第5擒：挣扎
```

## 第2擒 — 泸水天险

### 战役信息

| 项目 | 内容 |
|------|------|
| 场景 | 泸水河岸，激流中的木桩平台 |
| 色调 | 深蓝 + 灰白 |
| 预估时长 | 10-12 分钟 |
| 难度 | ★★☆☆☆ |
| 核心机制 | 跳跃挑战，水流障碍，浮水平台 |

### 开场叙事（图文过场）

> **[画面]** 汉军抵达泸水岸边，河水咆哮，对岸蛮营隐约可见。
>
> **诸葛亮**（对玩家）：「孟获退守泸水，以为天险可恃。然水流虽急，人心更急。」
>
> **AI 动态台词**（根据上关表现）：
> - 若上关激进速通 → 「上阵你锐不可当，然水战中机变比蛮力更重要。留意平台移动的规律。」
> - 若上关稳健 → 「步步为营，此乃良将之风。泸水之险不在水，在不识水性。」
>
> **[画面]** 汉军开始搭建浮桥。

### 关卡流程

| 阶段 | 内容 | 叙事点 |
|------|------|--------|
| 阶段1 | 浮桥推进，对抗水性蛮兵 | 对话框：孟获在远处嘲笑「过了泸水再说！」 |
| 阶段2 | 水上平台跳跃，急流区 | 对话框：诸葛亮观察地形，派人绕后 |
| 阶段3 | Boss战：孟获乘船而战 | 孟获台词变化，开始认真 |

### Boss战叙事

**孟获（乘战船）：**
- 攻击方式：船上弩箭齐射 / 撞击（船撞平台）/ 跳帮近战
- 新增机制：水面波动影响玩家站位
- 孟获（血量过半）：「泸水天险竟被尔等渡过……但你我的胜负，还未见分晓！」

### 擒获叙事（图文过场）

> **[画面]** 战船被击沉，孟获落水被汉军救起——注意：是"救起"。
>
> **孟获**（浑身湿透，但眼神不减）：「此乃地利之失，非战之罪！你若敢再放我……」
>
> **诸葛亮**：「天时不如地利，地利不如人和。将军请——」
>
> **[画面]** 诸葛亮递上一件干衣。孟获愣住了。
>
> **诸葛亮**：「南中潮湿，将军保重身体。」
>
> **孟获**（接过衣服，低声）：「……你究竟想要什么？」

### 孟获心态

> **阶段：辩解**
> ——「汉人占了地利便宜……但那个诸葛亮，为什么给我衣服？」

>
> **[画面]** 诸葛亮转身对部落首领们说话，而非只对孟获。
>
> **诸葛亮**：「诸位都是南中的勇士。但年年征战，部落死了多少男儿？孩子还有父亲吗？」
>
> **[画面]** 几个部落首领低下了头。木鹿大王坐在地上，沉默不语。
>
> **孟获**（声音变小了，但仍在坚持）：「……与汉人共治？可能吗？」

### 孟获心态

> **阶段：挣扎**

## v1.1 关卡叙事统一模板

每一关都应遵循同一套“战斗—擒获—释放—再战理由”的结构，但每次的情绪、机制和问题必须不同。

```
1. 开场：说明孟获这次为什么还能再战
2. 推进：用关卡机制表现孟获当前心态
3. Boss：让孟获的新理由变成具体招式
4. 擒获：让孟获提出“不服”的具体依据
5. 释放：诸葛亮回应，但不直接说教过满
6. 余味：玩家或孟获留下一个新的问题
```

### 每关必须回答的问题

| 关卡 | 必须回答的问题 | 不能重复的点 |
|------|----------------|--------------|
| 第1擒 | 为什么诸葛亮要活捉孟获？ | 不能把孟获写成只会骂人的反派 |
| 第2擒 | 孟获为什么认为自己只是输在地利？ | 不能重复“汉人狡诈” |
| 第3擒 | 孟获为什么开始困惑？ | 不能让诸葛亮直接说出全部答案 |
| 第4擒 | 孟获为什么明知会败仍坚持？ | 不能把祝融夫人写成附属品 |
| 第5擒 | 部落联盟为什么是孟获最大压力？ | 不能只做战斗堆怪 |
| 第6擒 | 木兽失败为何能触发反思？ | 不能把机关写成笑话 |
| 第7擒 | 孟获为什么主动降，而不是被迫降？ | 不能让最后一战只是数值更高 |

- 重玩时可跳过，但保留一键回看。

- AI 台词只出现在开场或结算，不插入关键剧情。
- 若生成失败，使用 GDD 中的 fallback 模板。
