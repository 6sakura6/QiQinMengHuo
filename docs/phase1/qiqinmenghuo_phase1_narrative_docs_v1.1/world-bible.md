## 1. 历史锚点

### 1.1 时间线

```
公元 223 年 — 刘备白帝城托孤，刘禅继位，诸葛亮受遗诏辅政
公元 225 年 — 南中（今云南、贵州、四川南部）蛮族叛乱，诸葛亮率军南征
         春 — 汉军从成都出发，沿岷江南下
         夏 — 进入南中腹地，与孟获交锋
         秋 — 七擒七纵完成，孟获归降
         冬 — 班师回成都
```

> 我们的游戏聚焦于公元 225 年夏秋之间，也就是七擒七纵发生的核心时段。

### 1.2 史书记载依据

**《三国志·蜀书·诸葛亮传》裴松之注引《汉晋春秋》：**

> "亮至南中，所在战捷。闻孟获者，为夷、汉所服，募生致之。既得，使观于营阵之间，问曰：'此军何如？'获对曰：'向者不知虚实，故败。今蒙赐观看营阵，若只如此，即定易胜耳。'亮笑，纵使更战。七纵七禽，而亮犹遣获。获止不去，曰：'公，天威也，南人不复反矣。'"

**历史还原度取舍原则：**

| 层面 | 策略 |
|------|------|
| 核心事件 | **严格遵循**：七擒七纵的历史框架、孟获最终折服 |
| 具体战斗 | **艺术加工**：史书未记载每场战斗细节，可自由设计关卡 |
| 蛮夷文化 | **立体呈现**：展现南中多部落文化，避免「蛮夷=反派」的刻板 |
| 人物性格 | **基于史实延伸**：诸葛亮智慧宽厚、孟获勇猛倔强均有史书支撑 |
| 台词 | **整体创作**：三国对话非史书原文，以文言白话融合呈现 |

### 2.3 核心冲突

**不是「文明 vs 野蛮」，而是「统一 vs 自治」。**

- 蜀汉视角：南中是蜀汉后方，必须稳定才能北伐中原
- 蛮族视角：南中是祖辈生活的土地，不想被外来政权统治
- **表层冲突**：军事对抗（横版射击的主体）
- **深层冲突**：两种治理理念的碰撞（叙事驱动的内核）

## 4. 「七擒七纵」的文化内核

### 4.1 为什么是「七」？

「七」在中国文化中是特殊数字：
- 天地之数：阴阳五行之和（2+5=7）
- 周期循环：七日来复，周而复始
- 完整性：七擒暗示了「完整的过程」
- **游戏化**：天然适合做7关解锁结构

### 4.2 「擒而不杀」的深层逻辑

诸葛亮选择七擒七纵，不是软弱的仁慈，而是最高明的战略：

```
直接歼灭的问题：
  ✗ 南中部落会推举新首领继续反抗
  ✗ 汉军无法长期驻守南中（北伐需要兵力）
  ✗ 屠杀会激发更大仇恨

七擒七纵的优势：
  ✓ 让孟获本人认识到实力差距
  ✓ 让所有部落看到汉军的克制和诚意
  ✓ 孟获折服后，整个南中自然归附
  ✓ 汉军可以安心撤军北伐
```

> **「攻心为上，攻城为下；心战为上，兵战为下。」**
> — 这不仅是诸葛亮的名言，也是我们游戏的核心价值主张。

## 7. v1.1 叙事伦理与表达边界

### 7.1 核心表达立场

本作的历史背景发生在三国时期，但面向现代玩家。创作时要明确区分“古代文献中的称谓”与“游戏表达中的价值判断”。游戏可以使用“南中”“部落”“蛮王”等历史语境词，但不应把南中人物写成低智、低级、只会嚎叫的反派。

| 可使用 | 使用方式 | 避免方式 |
|--------|----------|----------|
| 蛮王 | 历史语境中的称号，突出孟获为联盟领袖 | 不用于嘲笑、贬低、矮化 |
| 蛮兵 | 关卡敌人类型的功能称谓 | 不表现为无脑怪物 |
| 南中部族 | 强调地域、文化、自治诉求 | 不笼统写成“野人” |
| 归心 | 政治与心理层面的和解 | 不写成单方面征服和驯服 |

### 7.2 文化呈现四原则

1. **有秩序**：部落有首领、仪式、盟约、工匠和战术，不是混乱群体。  
2. **有理由**：孟获反抗是出于自治与族人安全，不是天生好战。  
3. **有尊严**：即使被击败，也保留战士荣誉和领袖责任。  
4. **有变化**：不同部落有不同地理、技术和信仰，不做单一模板复制。

### 7.3 诸葛亮表达边界

诸葛亮可以展现智慧，但不应无所不知。更好的写法是：

| 不推荐 | 推荐 |
|--------|------|
| “我早已算到一切。” | “我观其营阵，知其急于求胜。” |
| “孟获必降。” | “他重义守诺，若能见我军本心，终会明白。” |
| “蛮夷不识教化。” | “南中人心未安，非兵威可久定。” |

## 8. 世界观主题深化

### 8.1 表层冲突与深层冲突

| 层级 | 冲突 | 玩家感知 |
|------|------|----------|
| 表层 | 汉军与孟获军队作战 | 横版射击、Boss 战、敌兵推进 |
| 中层 | 蜀汉治理与南中自治的冲突 | 孟获每次不服都有新的理由 |
| 深层 | “以力服人”与“以心服人”的冲突 | 七擒七纵的真正意义逐步揭晓 |

### 8.2 诸葛亮为什么必须平定南中

游戏中应让玩家理解：诸葛亮不是为了个人功名南征，而是因为蜀汉北伐前必须稳定后方。南中若长期叛乱，蜀汉将陷入两线消耗；直接杀戮又会制造新的仇恨。因此“七擒七纵”是一种成本更高但后患更小的战略选择。

### 8.3 孟获为什么不能轻易投降

孟获不是一个人，他背后站着南中各部落。如果第一次战败就降，他会失去部落信任；如果不战到底，他又会让族人继续死亡。因此他的七阶段心理变化，本质是“个人尊严、领袖责任、族群未来”三者之间的撕扯。

## 12. 结局文本优化建议

原结局可以保留，但建议在最终版本中避免“汉军与蛮兵放下兵器”过于简单，增加“部落首领相互观望后才放下武器”的层次，让归心显得来之不易。

建议演出：

```
孟获放下战斧。
木鹿大王迟疑片刻，也放下木鞭。
董荼那看向身后的泸水部战士，点头。
祝融夫人站在阴影中，没有跪拜，只熄灭了掌中的火。
汉军士卒先看诸葛亮，诸葛亮轻轻摇扇，示意收弩。
两边的弓弩同时垂下。
```

这样既保留“南人不复反矣”的历史结论，也让各部落的尊严得到保留。
