## 总体修订方向

本次修订围绕“让概念更可执行、让叙事更闭环、让后续开发更容易落地”三个目标展开。原文档已经具备清晰的方向：像素三国、横版射击、七擒七纵、无名士卒视角、诸葛亮攻心、孟获心态递进。v1.1 在此基础上补强了以下内容：

1. 增加设计支柱与非目标，避免项目范围失控。
2. 明确 Phase 2 MVP 只做第 1 关完整纵切。
3. 增补 AI 动态台词系统的数据输入、输出限制、Prompt 与 fallback 模板。
4. 强化世界观表达边界，避免把南中部族脸谱化。
5. 将七关叙事拆成可转化为关卡脚本的触发节点。
6. 增补主要角色的台词规范、内心弧线、演出与资产建议。
7. 增加 QA 检查清单，便于后续团队自检。

## 本次修订原则

- 不改变原项目定位，只增强表达与落地性。
- 不新增会大幅扩大开发量的系统。
- 不把叙事写成单纯文本，而是尽量绑定机制、演出、数据结构和验收标准。
- 保留“七擒七纵”的历史核心，同时强化现代玩家可理解的角色动机。
