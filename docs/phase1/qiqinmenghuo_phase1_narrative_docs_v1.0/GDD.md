# 《七擒孟获》Game Design Document (GDD)

> 腾讯云黑客松 · 叙事类游戏赛道
> 版本：v1.0 | 日期：2026-06-11
> 阶段：Phase 1 — 产品定义 & 叙事设计

---

## 1. 产品 Identity

### 1.1 一句话描述

**一款像素三国风格的横版动作游戏，以"擒而不杀、纵而再战"的七擒七纵循环为叙事引擎，让玩家扮演汉军无名士卒，在诸葛亮的计谋指引下，经历七场与蛮王孟获的宿命对决。**

### 1.2 核心体验主张

| 维度 | 定义 |
|------|------|
| **战斗感** | 魂斗罗式横版射击，爽快直接，像素子弹横飞 |
| **叙事感** | 七擒七纵的循环结构，每场 Boss 战结束才是故事的开始 |
| **文化感** | 三国智慧 + 蛮夷文化的立体呈现，非标签式古装 |
| **成长感** | 孟获从傲慢到折服的心态弧线，玩家亲身见证 |

### 1.3 差异化定位

> **大多数动作游戏的 Boss 打倒即结束。《七擒孟获》的独特性在于：打倒 Boss 只是故事的开始。**

- **"擒而不杀"机制**：每次战胜孟获后，诸葛亮下令释放，触发叙事节点
- **"纵而再战"递进**：下次相遇时孟获更强、更狡猾，关卡机制同步深化
- **小兵视角**：无名汉军士卒的视角带来更多沉浸和共情空间
- **AI 计谋台词**：接入腾讯混元大模型，根据玩家行为动态生成诸葛亮谋略分析

## 2. 核心机制设计

### 2.1 战斗系统

| 操作 | 描述 | 设计参考 |
|------|------|---------|
| 移动 | 左右横移，标准平台移动 | 魂斗罗 |
| 跳跃 | 单段跳 + 空中微调 | 魂斗罗 |
| 射击 | 八方向射击（上下左右 + 斜向） | 魂斗罗 / 合金弹头 |
| 闪避 | 短无敌帧翻滚（第2关解锁） | 死亡细胞 |

**武器系统（渐进解锁）：**
- **基础弩箭**（默认）：直线射击，中速
- **连弩**（第3关解锁）：三连发，射速快但单发伤害低
- **火箭**（第5关解锁）：点燃敌人，持续伤害
- **诸葛连弩**（第7关）：全方向散射，伤害最高

### 2.2 叙事系统

```
关卡流程 = 开场叙事(图文过场) → 关卡战斗(对话框穿插) → Boss战 → 擒获叙事(图文过场) → 释放抉择 → 过关解锁
```

**叙事介入方式（图文过场 + 对话框）：**

| 节点 | 形式 | 时长 | 可跳过 |
|------|------|------|--------|
| 每关开场 | 图文过场（1-2屏） | 15-30秒 | ✅ |
| 关卡中途 | 对话框（右上角弹出） | 不打断战斗 | ✅ |
| Boss战前 | 对话框（孟获台词） | 2-3秒 | ✅ |
| 擒获后 | 图文过场（2-4屏） | 30-60秒 | ⚠️首通不可跳 |
| 诸葛亮释放令 | 对话框 + 动画 | 10-15秒 | ✅ |

### 2.3 故事碎片收集系统

- 每关隐藏 1 块「历史碎片」
- 集齐 7 块 → 解锁完整史书记载 + 《出师表》相关段落
- 增加重玩价值，连接历史真实

### 2.4 AI 动态计谋系统（加分项）

```
玩家上关数据 → 行为分析（激进/稳健/快/慢/死亡次数）
              → 混元大模型生成诸葛亮计谋台词
              → 下关开场过场中展示
```

## 5. 画风定义：「像素三国」

### 5.1 核心风格

- **基底**：像素风（16×16 或 32×32 Sprite），保持魂斗罗式的爽快感
- **国风点缀**：
  - UI 边框：水墨笔触纹理
  - 标题/关卡名：毛笔手写字体
  - 过场插画：水墨渲染 + 像素角色
  - 粒子效果：墨滴飞溅替代传统火花
- **调色板**：低饱和 + 朱砂红 / 墨绿 / 古铜为主色调

### 5.2 各关环境色调

| 关卡 | 场景 | 主色调 | 氛围关键词 |
|------|------|--------|-----------|
| 第1擒 | 西洱河平原 | 翠绿 + 天蓝 | 开阔、初战、轻敌 |
| 第2擒 | 泸水河岸 | 深蓝 + 灰白 | 险峻、湍急、沉着 |
| 第3擒 | 紫荆山密林 | 墨绿 + 褐黄 | 压抑、伏击、智斗 |
| 第4擒 | 秃龙洞洞窟 | 暗紫 + 荧光绿 | 诡异、毒雾、坚持 |
| 第5擒 | 蛮族联盟营寨 | 赭红 + 土黄 | 宏大、联战、暗流 |
| 第6擒 | 蛮夷要塞 | 深红 + 铁灰 | 烈火、机关、反制 |
| 第7擒 | 银坑洞王庭 | 金 + 深棕 | 壮阔、决战、折服 |

## 6. 玩家体验目标

### 6.1 情感曲线

```
第1关：自信（这蛮王不过如此）
第2关：警觉（地形开始变难了）
第3关：压力（蛮兵变狡猾了）
第4关：挣扎（毒雾让人窒息）
第5关：被压迫（双Boss好难）
第6关：翻盘（我找到了克制方法）
第7关：敬畏（孟获全力以赴，值得尊敬）
结局：感慨（七擒七纵，心战为上）
```

### 6.2 首关体验承诺

- 玩家在 30 秒内学会移动+跳跃+射击
- 2 分钟内遇到第一个叙事节点（诸葛亮界面出现）
- 5 分钟内完成首次 Boss 战并触发"释放"叙事
- 完整流程 ≈ 8-10 分钟
- 目标：第一关结束后玩家必须感到好奇——「为什么要放他走？」

## 7. 叙事 × 玩法的融合设计

### 7.1 核心原则

> **每关的机制变化反映孟获的心态变化。**

```
孟获心态变化     →    关卡机制变化
────────────────────────────────────
第1擒：轻视汉军   →    基础教学关，孟获几乎不认真
第2擒：开始布局   →    引入环境障碍（水流）
第3擒：调兵遣将   →    敌兵种类增加，伏击机制
第4擒：联合外援   →    毒雾环境伤害，考验资源管理
第5擒：倾巢而出   →    多波次 + 双Boss
第6擒：出奇制胜   →    机关陷阱 + 巨型兵器
第7擒：背水一战   →    全机制综合，三阶段Boss
```

### 7.2 擒获后的叙事变体

每次擒获孟获后，叙事重点不同：

| 关卡 | 孟获台词核心 | 诸葛亮回应核心 | 孟获心态 |
|------|------------|--------------|---------|
| 第1擒 | "诡计！汉人狡诈！" | "将军勇则勇矣，然未识天时。" | 愤怒、不服 |
| 第2擒 | "此乃地利之失，非战之罪！" | "天时不如地利，地利不如人和。" | 辩解、动摇 |
| 第3擒 | "……汝究竟欲如何？" | "欲将军心悦诚服耳。" | 困惑、开始理解 |
| 第4擒 | "纵我七次，我也不降！" | "七乃天地之数，将军且看。" | 倔强、但开始动摇 |
| 第5擒 | "我蛮族男儿，岂能降汉！" | "非降也，同心也。" | 挣扎、部落压力 |
| 第6擒 | "吾之木兽尚且不敌……" | "器不如人，非器之过。" | 技术碾压后的反思 |
| 第7擒 | "……亮公，孟获愿降。" | "将军非降亮，乃降于天下人心。" | 彻底折服 |

## 8. 关键指标

| 指标 | 目标 | 测量方式 |
|------|------|---------|
| 首关完成率 | > 90% | 关卡通关数据 |
| 全7关完成率 | > 50% | 关卡通关数据 |
| 平均单关时长 | 8-12 分钟 | 计时统计 |
| 对话框跳过率 | < 40%（首次） | 交互统计 |
| 故事碎片收集率 | > 30% | 碎片获取统计 |
| 帧率 | 稳定 60fps | 性能监控 |
| 首屏加载 | < 5 秒 | 加载计时 |
