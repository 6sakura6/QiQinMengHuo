## 叙事主线

```
诸葛亮率汉军南征 → 七次擒获孟获 → 七次释放 → 孟获从傲慢到折服 → 南中归心

核心叙事钩子：为什么打赢了还要放人？ → 答案随7关逐步揭晓
```

## 孟获心态变化弧线

```
第1擒：傲慢 ──→ 第2擒：辩解 ──→ 第3擒：困惑 ──→ 第4擒：倔强
                                                ↓
第7擒：折服 ←── 第6擒：反思 ←── 第5擒：挣扎
```

## 第1擒 — 西洱河初战

### 战役信息

| 项目 | 内容 |
|------|------|
| 场景 | 南中平原，西洱河畔，开阔草地 |
| 色调 | 翠绿 + 天蓝 |
| 预估时长 | 8-10 分钟（教学关） |
| 难度 | ★☆☆☆☆ |
| 核心机制 | 基础移动/跳跃/射击教学 |

### 开场叙事（图文过场）

> **[画面]** 汉军旌旗在平原上猎猎作响，诸葛亮立于中军帐前，遥望南中群山。
>
> **诸葛亮**：「南中不平，北伐无以为继。此战所求，非杀戮，乃归心。」
>
> **[画面]** 军帐中，诸葛亮将一卷竹简递给身旁侍立的年轻士卒（玩家）。
>
> **诸葛亮**：「你随我出征以来，屡立战功。此去西洱河，孟获必轻敌冒进。记住——」
>
> **诸葛亮**：「我要活的。」

### 关卡流程

| 阶段 | 内容 | 叙事点 |
|------|------|--------|
| 阶段1 | 平原行军，遇零星蛮兵 | 对话框：老兵提醒"南中地势不熟，小心脚下" |
| 阶段2 | 河岸遭遇战，蛮兵增多 | 对话框：孟获斥候出现，不屑一顾 |
| 阶段3 | Boss战：孟获骑象冲锋 | 孟获台词：「汉人小儿，也敢犯我南中！」 |

### Boss战叙事

**孟获（骑战象）：**
- 攻击方式：象鼻横扫 / 踏地震击 / 冲锋
- 孟获台词（血量下降时）：「哼……有几分本事。但还不够！」
- 孟获（被击落象背）：「什么？！这不可能——」

### 擒获叙事（图文过场）

> **[画面]** 孟获被汉军士兵押至诸葛亮面前，昂首不跪。
>
> **孟获**：「诡计！汉人狡诈，我不服！若堂堂正正一战，胜负未可知！」
>
> **[画面]** 诸葛亮微笑，示意士兵松绑。
>
> **诸葛亮**：「将军勇则勇矣，然未识天时。请回营整军，改日再战。」
>
> **诸葛亮**（低声对玩家）：「一擒不足以服其心。你做得很好——去歇息吧。」
>
> **[画面]** 孟获震惊地回头看了诸葛亮一眼，大步离去。

### 孟获心态

> **阶段：傲慢**
> ——「汉人不过侥幸。下次我认真打，必叫他们有来无回。」

## 第2擒 — 泸水天险

### 战役信息

| 项目 | 内容 |
|------|------|
| 场景 | 泸水河岸，激流中的木桩平台 |
| 色调 | 深蓝 + 灰白 |
| 预估时长 | 10-12 分钟 |
| 难度 | ★★☆☆☆ |
| 核心机制 | 跳跃挑战，水流障碍，浮水平台 |

### 开场叙事（图文过场）

> **[画面]** 汉军抵达泸水岸边，河水咆哮，对岸蛮营隐约可见。
>
> **诸葛亮**（对玩家）：「孟获退守泸水，以为天险可恃。然水流虽急，人心更急。」
>
> **AI 动态台词**（根据上关表现）：
> - 若上关激进速通 → 「上阵你锐不可当，然水战中机变比蛮力更重要。留意平台移动的规律。」
> - 若上关稳健 → 「步步为营，此乃良将之风。泸水之险不在水，在不识水性。」
>
> **[画面]** 汉军开始搭建浮桥。

### 关卡流程

| 阶段 | 内容 | 叙事点 |
|------|------|--------|
| 阶段1 | 浮桥推进，对抗水性蛮兵 | 对话框：孟获在远处嘲笑「过了泸水再说！」 |
| 阶段2 | 水上平台跳跃，急流区 | 对话框：诸葛亮观察地形，派人绕后 |
| 阶段3 | Boss战：孟获乘船而战 | 孟获台词变化，开始认真 |

### Boss战叙事

**孟获（乘战船）：**
- 攻击方式：船上弩箭齐射 / 撞击（船撞平台）/ 跳帮近战
- 新增机制：水面波动影响玩家站位
- 孟获（血量过半）：「泸水天险竟被尔等渡过……但你我的胜负，还未见分晓！」

### 擒获叙事（图文过场）

> **[画面]** 战船被击沉，孟获落水被汉军救起——注意：是"救起"。
>
> **孟获**（浑身湿透，但眼神不减）：「此乃地利之失，非战之罪！你若敢再放我……」
>
> **诸葛亮**：「天时不如地利，地利不如人和。将军请——」
>
> **[画面]** 诸葛亮递上一件干衣。孟获愣住了。
>
> **诸葛亮**：「南中潮湿，将军保重身体。」
>
> **孟获**（接过衣服，低声）：「……你究竟想要什么？」

### 孟获心态

> **阶段：辩解**
> ——「汉人占了地利便宜……但那个诸葛亮，为什么给我衣服？」

| 难度 | ★★★☆☆ |
| 核心机制 | 视野遮挡，伏击蛮兵，识破隐身 |

### 开场叙事（图文过场）

> **[画面]** 密林深处，汉军小心翼翼推进。树影婆娑，处处危机。
>
> **诸葛亮**：「阿会喃善山林伏击。蛮兵藏于树冠，你需耳听八方。」
>
> **AI 动态台词**：
> - 若上关死亡较多 → 「前路艰险，然诸葛亮有一言相赠——'临事而惧，好谋而成'。慎之。」
> - 若上关顺利 → 「南中地势愈深愈险。不可因两胜而生骄。」

### 关卡流程

| 阶段 | 内容 | 叙事点 |
|------|------|--------|
| 阶段1 | 密林潜行，遭遇伏击 | 对话框：阿会喃的伏兵从树上落下 |
| 阶段2 | 视野转换区，间歇迷雾 | 对话框：汉军搜索蛮族营地 |
| 阶段3 | Boss战：孟获隐身偷袭 | 孟获利用丛林地形 |

### Boss战叙事

**孟获（丛林猎手模式）：**
- 攻击方式：隐身突袭 / 设置绊索陷阱 / 远程吹箭
- 新机制：需要观察环境线索（树叶抖动、脚印）预判孟获位置
- 孟获台词：「在我的地盘，你不过是猎物！」

### 擒获叙事（图文过场）

> **[画面]** 孟获从树上被射落，摔在地上，这次没有挣扎。
>
> **孟获**：「……设伏不成，反被识破。汝究竟欲如何？」
>
> **[画面]** 诸葛亮亲自扶起孟获。
>
> **诸葛亮**：「欲将军心悦诚服耳。」
>
> **孟获**（沉默良久）：「心悦诚服？呵……那你得先打败我的祝融夫人。」
>
> **[画面]** 诸葛亮眼中闪过一丝欣赏。
>
> **诸葛亮**：「好。请将军带路。」

### 孟获心态

> **阶段：困惑**
> ——「他明明可以杀我三次了……难道汉人真的不一样？」

## 第4擒 — 秃龙洞毒雾

### 战役信息

| 项目 | 内容 |
|------|------|
| 场景 | 秃龙洞地下洞窟，毒雾弥漫 |
| 色调 | 暗紫 + 荧光绿 |
| 预估时长 | 12-15 分钟 |
| 难度 | ★★★★☆ |
| 核心机制 | 环境伤害（毒雾），周期性视野黑暗，资源管理 |

### 开场叙事（图文过场）

> **[画面]** 洞窟入口，白雾缭绕，汉军士兵面露惧色。
>
> **诸葛亮**：「秃龙洞乃祝融氏领地，毒雾终年不散，外人入者九死一生。」
>
> **诸葛亮**（对玩家）：「但孟获既然选了这条路，说明他已无退路——这正是我们想要的。」
>
> **AI 动态台词**：
> - 「毒雾虽厉，火把可驱。每前一步，先寻光源。」
> - 「祝融夫人善用火攻和毒术。这两者有一个共同弱点——风。」

### 关卡流程

| 阶段 | 内容 | 叙事点 |
|------|------|--------|
| 阶段1 | 毒雾区域，寻找火把/光源 | 对话框：祝融夫人的警告回荡洞中 |
| 阶段2 | 黑暗区域，间歇照明 | 对话框：蛮兵利用黑暗偷袭 |
| 阶段3 | 最终洞窟 | Boss战前，祝融夫人现身 |

### Boss战叙事

**孟获 + 祝融夫人（未直接战斗）：**
- 攻击方式：孟获释放毒雾弹 / 召唤毒虫 / 洞窟塌方
- 新机制：视野周期性全黑，玩家只能靠火把短暂照明
- 祝融夫人台词（洞中回荡）：「敢入我洞者，皆为枯骨！」

### 擒获叙事（图文过场）

> **[画面]** 孟获靠在洞壁上，精疲力竭。洞窟深处，祝融夫人的身影一闪而逝。
>
> **孟获**（喘气）：「祝融已经走了……你追不上她的。」
>
> **诸葛亮**：「今日不为追人，只想与将军说句话。」
>
> **孟获**：「说。」
>
> **诸葛亮**：「将军每次被擒，身边部落越来越少。若再战下去——」
>
> **孟获**（打断）：「纵我七次，我也不降！」
>
> **[画面]** 诸葛亮静静看着孟获，眼中不是失望，而是理解和尊重。
>
> **诸葛亮**：「七乃天地之数，将军且看。」

### 孟获心态

> **阶段：倔强**
> ——「我不能降。降了，南中就不是南中了。但……他的眼神里没有轻蔑。」

## 第5擒 — 蛮族联军

### 战役信息

| 项目 | 内容 |
|------|------|
| 场景 | 蛮族联盟营地，宽阔战场 |
| 色调 | 赭红 + 土黄 |
| 预估时长 | 15-18 分钟 |
| 难度 | ★★★★☆ |
| 核心机制 | 多波次敌兵，双Boss战，精英蛮兵 |

### 开场叙事（图文过场）

> **[画面]** 蛮族联军大营，各部落旗帜招展。孟获站在高台上，召集了南中所有部落。
>
> **孟获**（对部落首领们）：「汉军已入我腹地！今日不是孟获一人的战争——是南中所有人的战争！」
>
> **木鹿大王**：「我木鹿部愿出木兽！」
> **朵思大王**：「机关已备，管教汉军有来无回！」
>
> **[画面]** 镜头切回汉军营地。
>
> **诸葛亮**：「孟获发动了南中所有能战之力。这是好事。」
>
> **玩家**（内心独白）：「好事？」
>
> **诸葛亮**：「只有倾尽全力后失败，才能真正服气。此战之后，他再无借口。」

### 关卡流程

| 阶段 | 内容 | 叙事点 |
|------|------|--------|
| 阶段1 | 联军前哨战，多种蛮兵轮换 | 对话框：各部落兵种差异化展示 |
| 阶段2 | 突破营寨防线，精英兵种 | 对话框：木鹿大王的木兽首次出现 |
| 阶段3 | Boss战：孟获 + 木鹿大王 | 双Boss，需同时应对 |

### Boss战叙事

**孟获 + 木鹿大王（双Boss）：**
- 木鹿大王：操控木兽，远程喷火 / 冲撞
- 孟获：近战猛攻，保护木鹿大王
- 特殊机制：击败木鹿大王后，孟获进入暴怒状态（伤害+速度提升）
- 孟获台词：「木鹿！不——！我跟你拼了！」

### 擒获叙事（图文过场）

> **[画面]** 联军营地一片狼藉。孟获被按在地上，周围的部落首领们垂头丧气。
>
> **孟获**（挣扎）：「我蛮族男儿，岂能降汉！」
>
> **诸葛亮**（走向孟获，语气平静）：「非降也，同心也。」
>
> **[画面]** 诸葛亮转身对部落首领们说话，而非只对孟获。
>
> **诸葛亮**：「诸位都是南中的勇士。但年年征战，部落死了多少男儿？孩子还有父亲吗？」
>
> **[画面]** 几个部落首领低下了头。木鹿大王坐在地上，沉默不语。
>
> **孟获**（声音变小了，但仍在坚持）：「……与汉人共治？可能吗？」

### 孟获心态

> **阶段：挣扎**
> ——「我不是不相信他。我只是不相信汉人。可……他连部落首领都放过。」

> **诸葛亮**：「木兽火攻，乃是朵思大王的机关术巅峰。然火攻最怕——」

> **诸葛亮**：「——以其人之道，还治其人之身。」
>
> **AI 动态台词**：

|------|------|--------|

**孟获（驾驶巨型木兽）：**

- 特殊机制：需要引诱木兽攻击燃料桶，引发爆炸伤害木兽自身
- 孟获台词：「来吧！这是南中最后的机关！」
- 木兽被毁后：「朵思的机关……竟被如此破解……」

### 擒获叙事（图文过场）

> **[画面]** 木兽残骸冒着黑烟。孟获从驾驶舱中跌出，满脸灰烬。
>
> **孟获**：「吾之木兽尚且不敌……诸葛亮，你是妖魔吗？」
>
> **[画面]** 诸葛亮走过来，伸出手。
>
> **诸葛亮**：「器不如人，非器之过。将军可知我为何每次放你？」
>
> **孟获**：「……为了羞辱我？」
>
> **诸葛亮**（摇头）：「为了让将军明白——汉军来南中，不是来征服的。」
>
> **[画面]** 孟获第一次没有立刻站起来离开。他坐在地上，抬头看着诸葛亮。
>
> **孟获**：「那你们来做什么？」
>
> **诸葛亮**：「来交一个朋友。」
>
> **[画面]** 沉默。只有余火噼啪作响。

### 孟获心态

> **阶段：反思**
> ——「他不是妖魔。他比妖魔更难对付——因为他说的都是真的。」

> **[画面]** 银坑洞深处，孟获独自站在王庭中央。祖先的灵位排列两侧，火光摇曳。
>

>
> **孟获**：「但今日是我最后的战斗。不为胜负，为南中男儿的尊严。」
>
> **[画面]** 汉军营中，诸葛亮整理衣冠，神色前所未有地庄重。
>

### Boss战叙事 — 全游戏最重要的战斗

**孟获（三阶段变身）：**

**第一阶段：猛将孟获**
- 前6关所有基础攻击模式强化版
- 快速、凶猛、毫不犹豫
- 台词：「这是我最后的力量！接招！」

**第二阶段：蛮王孟获**
- 血条归零后变身，身体被祖灵之力包裹
- 招式融合所有前6关机制：毒雾、隐身、冲锋、木兽残骸攻击
- 台词：「列祖列宗！赐我力量！」

**第三阶段：凡人孟获**
- 所有力量耗尽，回到最初的模样
- 不再攻击，而是向玩家走来——步行，不奔跑
- **这是叙事设计的核心：Boss 不再战斗**
- 孟获走到玩家面前，放下武器

