## 1. 历史锚点

### 1.1 时间线

```
公元 223 年 — 刘备白帝城托孤，刘禅继位，诸葛亮受遗诏辅政
公元 225 年 — 南中（今云南、贵州、四川南部）蛮族叛乱，诸葛亮率军南征
         春 — 汉军从成都出发，沿岷江南下
         夏 — 进入南中腹地，与孟获交锋
         秋 — 七擒七纵完成，孟获归降
         冬 — 班师回成都
```

> 我们的游戏聚焦于公元 225 年夏秋之间，也就是七擒七纵发生的核心时段。

### 1.2 史书记载依据

**《三国志·蜀书·诸葛亮传》裴松之注引《汉晋春秋》：**

> "亮至南中，所在战捷。闻孟获者，为夷、汉所服，募生致之。既得，使观于营阵之间，问曰：'此军何如？'获对曰：'向者不知虚实，故败。今蒙赐观看营阵，若只如此，即定易胜耳。'亮笑，纵使更战。七纵七禽，而亮犹遣获。获止不去，曰：'公，天威也，南人不复反矣。'"

**历史还原度取舍原则：**

| 层面 | 策略 |
|------|------|
| 核心事件 | **严格遵循**：七擒七纵的历史框架、孟获最终折服 |
| 具体战斗 | **艺术加工**：史书未记载每场战斗细节，可自由设计关卡 |
| 蛮夷文化 | **立体呈现**：展现南中多部落文化，避免「蛮夷=反派」的刻板 |
| 人物性格 | **基于史实延伸**：诸葛亮智慧宽厚、孟获勇猛倔强均有史书支撑 |
| 台词 | **整体创作**：三国对话非史书原文，以文言白话融合呈现 |

### 2.3 核心冲突

**不是「文明 vs 野蛮」，而是「统一 vs 自治」。**

- 蜀汉视角：南中是蜀汉后方，必须稳定才能北伐中原
- 蛮族视角：南中是祖辈生活的土地，不想被外来政权统治
- **表层冲突**：军事对抗（横版射击的主体）
- **深层冲突**：两种治理理念的碰撞（叙事驱动的内核）

## 3. 蛮夷文化设定

### 3.1 设计原则

> **蛮夷文化有其内在逻辑和尊严。非单纯反派，而是价值观不同的另一方。**

### 3.2 部落设定

| 部落 | 首领 | 领地 | 特点 | 对应关卡 |
|------|------|------|------|---------|
| 孟氏本部 | 孟获 | 银坑洞 | 南中最大部落，善骑射 | 第1/7擒 |
| 泸水部 | 董荼那 | 泸水沿岸 | 水性精熟，善水战 | 第2擒 |
| 紫荆山部 | 阿会喃 | 紫荆山密林 | 善伏击、丛林陷阱 | 第3擒 |
| 祝融氏 | 祝融夫人 | 秃龙洞 | 用毒、火攻，女性首领 | 第4擒 |
| 木鹿部 | 木鹿大王 | 联盟营地 | 驯兽、驱使木兽 | 第5/6擒 |
| 朵思部 | 朵思大王 | 要塞 | 工匠、机关术 | 第6擒 |

### 3.3 蛮族文化核心价值

| 价值 | 表现 | 游戏中的体现 |
|------|------|------------|
| **勇武** | 崇尚武力，以战死为荣 | Boss 战孟获越战越勇 |
| **义气** | 部落之间以血誓联盟 | 第5擒双Boss的协作 |
| **祖灵信仰** | 敬畏自然和祖先 | 洞窟场景中的图腾柱、祭祀火堆 |
| **独立精神** | "不自由毋宁死" | 孟获反抗的核心动机 |
| **重诺守信** | 一旦认输则永不反叛 | 第7擒后彻底折服 |

### 3.4 蛮族视觉语言

```
视觉元素清单：
├── 图腾柱：木雕彩绘，狼/虎/鹰图腾
├── 战鼓：牛皮蒙面，战前鼓舞士气
├── 骨饰：兽骨项链、角盔，非野蛮而是荣耀象征
├── 纹面：部落战士的面部彩绘，不同部落图案不同
├── 营寨：竹木结构，与自然共生而非破坏
└── 祭祀火：祖灵信仰的核心仪式
```

## 4. 「七擒七纵」的文化内核

### 4.1 为什么是「七」？

「七」在中国文化中是特殊数字：
- 天地之数：阴阳五行之和（2+5=7）
- 周期循环：七日来复，周而复始
- 完整性：七擒暗示了「完整的过程」
- **游戏化**：天然适合做7关解锁结构

### 4.2 「擒而不杀」的深层逻辑

诸葛亮选择七擒七纵，不是软弱的仁慈，而是最高明的战略：

```
直接歼灭的问题：
  ✗ 南中部落会推举新首领继续反抗
  ✗ 汉军无法长期驻守南中（北伐需要兵力）
  ✗ 屠杀会激发更大仇恨

七擒七纵的优势：
  ✓ 让孟获本人认识到实力差距
  ✓ 让所有部落看到汉军的克制和诚意
  ✓ 孟获折服后，整个南中自然归附
  ✓ 汉军可以安心撤军北伐
```

> **「攻心为上，攻城为下；心战为上，兵战为下。」**
> — 这不仅是诸葛亮的名言，也是我们游戏的核心价值主张。

## 5. 历史改编策略

### 5.1 保留的史实要素

- 诸葛亮南征的整体背景和动机
- 七擒七纵的基本框架
- 孟获作为蛮族领袖的地位
- 南中地理的基本特征
- 「南人不复反」的历史结局

### 5.2 艺术加工的部分

| 加工方向 | 具体操作 | 目的 |
|---------|---------|------|
| 战斗细节 | 将史书中一笔带过的战斗扩展为7个关卡 | 填充游戏内容 |
| 角色深化 | 给孟获更多台词和情感层次 | 增强叙事感染力 |
| 蛮夷文化 | 虚构部落命名和风俗（基于西南少数民族原型） | 增加文化厚度 |
| 无名主角 | 虚构汉军士卒视角 | 增强玩家代入感 |
| AI台词 | 诸葛亮动态计谋分析 | 技术创新加分 |

### 5.3 有意识避开的雷区

| 雷区 | 替代方案 |
|------|---------|
| 将蛮夷描绘为未开化的野蛮人 | 展现其文化逻辑和内在尊严 |
| 诸葛亮全知全能的「神化」 | 展现其智慧和战略眼光，但保留人性 |
| 简单粗暴的「正义征服邪恶」 | 呈现为两种价值观的碰撞与和解 |
| 女性角色缺失 | 祝融夫人作为独立强大角色出现 |
