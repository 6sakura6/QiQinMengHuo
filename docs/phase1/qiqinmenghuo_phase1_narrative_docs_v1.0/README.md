## 核心叙事设计亮点

1. **「擒而不杀」循环**：Boss 战结束 = 叙事开始，这在整个游戏行业是独特的
2. **孟获心态弧线**：傲慢 → 辩解 → 困惑 → 倔强 → 挣扎 → 反思 → 折服（七阶递进）
3. **小兵视角**：汉军无名士卒的视角让「七擒七纵」从宏大叙事变为个人体验
4. **AI 动态台词**：接入混元大模型，根据玩家行为生成诸葛亮计谋分析
5. **色调叙事**：七关色调从翠绿到金棕，反映孟获心态逐步沉淀
